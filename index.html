Mais sobre mim

Titulos servem para definir o tema de um site. Existem seis níveis de títulos

Podemos destacar trechos de texto para enfatizar uma idéia

  • Uma lista não ordenada não se importa com a ordem
  • UL significa "unordered list"
  • LI significa List Item
  1. Uma lista ordenada é uma lista que possue algum tipo de ordem numérica
  2. Seu funcionamento é exatamente como uma ul
  3. OL significa "Ordered List"
